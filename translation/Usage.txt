Usage 사용법

개인적으로 이러한 예시들을 최대한 활용할 수 있는 좋은 방법은 시간 순서대로 모든 예제를 읽는 것입니다.

예제를 설정하기 위해 초기 코드를 주의 깊게 읽어야 합니다. 만약 당신이 숙련된 Phyton 프로그래머라면, 대부분의 경우 당신은 다음에 무슨 일이 일어날지 성공적으로 예측할 수 있을 것입니다.
출력된 결과를 읽고, 출력이 예상과 동일한지 확인하십시오.
출력 정보가 생성된 정확한 이유를 알고 있는 지 확인하십시오.
만약 그렇지 않다면, 심호흡을 하고 설명을 다시 읽으십시오. (만약 그래도 이해되지 않는다면 크게 소리진 후 이 곳에 이슈를 올려주십시오 ^^)
만약 그렇다면, 등을 가볍게 두드리며 다음 예제로 넘어가셔도 됩니다. 

PS : 명령줄에서 WTFpython도 읽을 수 있습니다. pypi 패키지와 색상의 포맷을 지원하는 npm 패키지을 같은 용도로 활용할 수 있습니다.

npm 패키지 wtfpython을 설치하기 위해서는 아래와 같이 입력합니다.
$ npm install -g wtfpython

위의 명령어 대신 pypi 패키지 wtfpython을 설치할 수 있습니다.
$ pip install wtfpython -U

이제 당신의 $PAGER에서 이 컬렉션을 열기 위해 커맨드 라인에서 wtfpython을 실행합니다.
